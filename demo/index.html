About this project

                    

                        
Every base model writes 2023-era AI code — ask one for OpenAI and you get APIs deleted years ago. genai-coder is Qwen2.5-Coder-7B trained out of that habit on 6,665 question→code pairs whose answers are all current, human-written example code harvested from permissively-licensed SDK repos and cookbooks. A local model wrote the questions; humans wrote every answer.

                        
Training ran overnight on this machine's RTX A2000 (12 GB): QLoRA rank-16 adapters over the 4-bit base, 748 steps in about five hours, peak 9.8 GB VRAM, final train loss 0.59. The merged model was quantized to GGUF Q4_K_M (4.7 GB) and registered with Ollama — the same server answering this page right now.

                        
On a 50-prompt SDK-currency benchmark the stock model scores 12%; this fine-tune scores 72%. In the failure analysis, not one of its answers used a deprecated API — the habit it was trained against is gone.

                        
The finale: driven by a small agent harness, the model wrote this chat app's first version itself — files, verification, and all — and Build mode still does that live: it writes code to disk, runs it, reads the errors, fixes them, and proves servers work with real HTTP requests. Everything happens on this machine. Nothing leaves it.
